Analyze an RFP Beginner

15-20 minutes

Scenario

You've just received an RFP from Sunrise Furniture Co. They're looking for a 3PL partner. Your sales manager wants a quick summary of what they're asking for and whether we should pursue it.

Open the RFP Analysis Prompt

Go to the AI Prompts section and open "RFP Analysis Prompts"

Open RFP Analysis Prompts

Copy "Prompt 1: Basic RFP Extraction"

This prompt will extract key requirements from the RFP into a structured summary.

Get the Sample RFP

Open the sample RFP and copy its contents.

Open Sample RFP

Paste into ChatGPT or Claude

Paste the prompt, then paste the RFP content where it says "[PASTE RFP HERE]"

Review the Output

The AI should give you a structured summary with customer info, services requested, volumes, timeline, and questions to ask.

Success Criteria

  • Output identifies customer name and contact
  • Services requested are clearly listed
  • Volume estimates are extracted
  • Key dates and timeline are noted
  • Questions to ask are generated

Write a Delay Notification Intermediate

15 minutes

Scenario

A shipment is delayed due to a carrier breakdown. The customer (a 3-year account worth $45K/month) previously complained about not being notified promptly. You need to write a professional email that's empathetic but not groveling.

Open Customer Communication Prompts

Open Customer Communication Prompts

Copy "Scenario A: Shipment Delay Notification"

This prompt includes all the context about the customer relationship and situation.

Run and Evaluate

Does the email acknowledge their previous feedback? Is the tone right?

Try Adjusting

Ask the AI to make it shorter, or more apologetic, or include a discount offer. See how it adapts.

Success Criteria

  • Email is under 200 words
  • Acknowledges their previous feedback about communication
  • Provides new ETA clearly
  • Explains what you're doing about it
  • Tone is professional, not excessive

Pro Tip

Try running the same prompt in both ChatGPT and Claude. Compare the outputs - you'll notice different styles and approaches!

Scenario

Your team describes their quote process differently - it's messy and inconsistent. You need to document the actual process as a flowchart that can be used for training and improvement.

Open Process Mapping Prompts

Open Process Mapping Prompts

Get the Messy Process Description

Open Messy Process Description

Run and Get Mermaid Code

The AI will output Mermaid diagram code.

Visualize the Diagram

Go to mermaid.live and paste the code to see your flowchart.

Success Criteria

  • Mermaid code renders without errors
  • Flowchart shows logical process flow
  • Pain points are highlighted in red

Pro Tip

If the Mermaid code has errors, paste the error message back to the AI and ask it to fix the code. It will usually correct it on the second try.
